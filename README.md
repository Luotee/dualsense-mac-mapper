# DualSense Mapper

Two implementations live in this repo:

- `legacy-python/` — original Python POC (May 2025), still functional, kept for reference.
- `rust/` — Rust rewrite (in progress), targets a single Windows `.exe` for non-technical end users.

See `rust/README.md` for build and usage. Top-level README will be expanded once the Rust build ships.
