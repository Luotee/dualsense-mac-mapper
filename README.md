## Why this exists?
Background This project was born in May 2025. While playing MapleStory Worlds Artale with my wife, I looked for a controller mapping tool to save her from the discomfort of long sessions on a MacBook keyboard.

After testing multiple macOS mapping tools from both the App Store and GitHub, I found a recurring issue: keys would frequently get "stuck." This happened because the software registered the "button down" event but missed the "button up" signal. Frustrated after trying several unsuccessful solutions, I turned to ChatGPT and GitHub Copilot to build this tool from scratch. Remarkably, except for fine-tuning the DEADZONE and THRESHOLD parameters, the code was entirely generated by AI.
